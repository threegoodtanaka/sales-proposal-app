# 営業提案文作成アプリ

Next.js と Tailwind CSS で作った営業提案文の作成ツールです。

## 機能

- **フォーム**: 会社名・商品名・課題を入力
- **提案文を生成**: 入力内容から提案文を生成
- **表示・コピー**: 生成文を表示し、ワンクリックでコピー

## 起動方法

Node.js をインストールしたうえで、プロジェクトフォルダで以下を実行してください。

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

## 構成

- メインの画面・ロジックは `app/page.tsx` にまとまっています。
- レイアウト: `app/layout.tsx`
- スタイル: `app/globals.css`（Tailwind）
