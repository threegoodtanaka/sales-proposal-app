# チャットボットアプリ（中学生向けの説明）

このアプリは、あなたが書いたメッセージに、AI（ChatGPT）が返事をしてくれるチャットです。  
Flask（フラスク）という Python の道具と、ChatGPT の API を使っています。

---

## 使う前に用意するもの

1. **Python**  
   パソコンに Python が入っていること（バージョン 3.8 以上がおすすめ）。
2. **ChatGPT の API キー**  
   「AI に話しかけるためのパスワード」のようなものです。  
   お金がかかる場合があるので、大人の人と相談してから取ってね。

---

## やり方（3つのステップ）

### ステップ1：このアプリのフォルダに移動する

パソコンの「ターミナル」や「PowerShell」を開いて、次のように打ちます。

```powershell
cd X
```

※ `cd` は「このフォルダに移動する」という意味です。

---

### ステップ2：必要な道具を入れる

次のコマンドを打ちます。

```powershell
pip install -r requirements.txt
```

※ これは「このアプリを動かすために必要なプログラムを、まとめて入れる」という意味です。  
  少し時間がかかることがあります。

---

### ステップ3：API キーを教えておく

ChatGPT の API キーを、パソコンに「この名前で覚えておいて」と伝えます。

**PowerShell で打つとき（いちどだけ）：**

```powershell
$env:OPENAI_API_KEY = "ここにあなたのAPIキーを入れる"
```

※ `"sk-..."` のように、引用符 "" の中に、本当の API キーを入れます。  
※ API キーは [OpenAI のサイト](https://platform.openai.com/api-keys) で作れます（会員登録が必要です）。

---

## アプリの起動方法（ここが一番大事）

1. **まず、ステップ1の「cd X」をしたフォルダにいることを確認する。**

2. **次のコマンドを打つ：**

   ```powershell
   python app.py
   ```

3. **画面に「Running on http://127.0.0.1:5000/」のような表示が出たら成功です。**

4. **ブラウザ（Chrome や Edge など）を開いて、次のアドレスを入力する：**

   ```
   http://127.0.0.1:5000/
   ```

5. **チャットの画面が出たら、下の入力欄にメッセージを書いて「送信」を押す。**

- **あなたのメッセージ** → 青っぽい色で表示されます。  
- **AI の返事** → 緑っぽい色で表示されます。  
  色で「自分」と「AI」がすぐわかります。

---

## まとめ（起動の流れ）

| 順番 | やること |
|------|----------|
| 1 | ターミナルで `cd X` を打つ |
| 2 | `pip install -r requirements.txt` を打つ（初回だけ） |
| 3 | `$env:OPENAI_API_KEY = "あなたのAPIキー"` を打つ（初回だけ） |
| 4 | `python app.py` を打つ |
| 5 | ブラウザで `http://127.0.0.1:5000/` を開く |

---

## 注意してほしいこと

- **API キーは人に教えたり、SNS に載せたりしないでください。** パスワードと同じです。
- API キーを設定していないと「OPENAI_API_KEY が設定されていません」というエラーが出ます。そのときはステップ3をもう一度やってみてください。

---

## ファイルの説明（おまけ）

- **app.py** … アプリの中心のプログラム。あなたのメッセージを ChatGPT に送って、返事をもらうところ。
- **templates/index.html** … チャットの画面のデザイン（青と緑の色分けもここで決めている）。
- **requirements.txt** … 「どんなプログラムを pip で入れればいいか」のリスト。
- **README.md** … 今読んでいるこの説明のファイル。
